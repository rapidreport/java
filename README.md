RapidReport/java
==================
RapidReport/java は Java 用の帳票レンダリングエンジンです。

JSON 形式で記述した帳票定義ファイルを読み込み、データを渡すことで帳票を出力します。

RapidReport  
[http://rapidreport.systembase.co.jp/](http://rapidreport.systembase.co.jp/)
